# App

Base estrutural do sistema fullstack orientado por schema, com frontend, backend e áreas compartilhadas separadas desde o início.

## Estrutura

- `frontend/` para a aplicação React.
- `backend/` para a API NestJS.
- `shared/` para contratos e ativos compartilhados no futuro.
- `docs/` para documentação de apoio.

## Diretriz da etapa

Esta fase cria apenas a base do repositório. Não há implementação funcional aqui, apenas organização clara para sustentar as próximas etapas sem acoplamento prematuro.
