# Frontend

Aplicação React + TypeScript que será evoluída em etapas pequenas.

## Estrutura prevista

- `src/app/` para bootstrap da aplicação.
- `src/components/` para biblioteca reutilizável.
- `src/features/` para módulos de domínio.
- `src/schemas/` para schemas de UI.
- `src/stores/` para Zustand.
- `src/services/` para integrações e persistência.
- `src/types/` para tipos compartilhados do frontend.

Nesta etapa, a pasta existe apenas como base organizacional.
