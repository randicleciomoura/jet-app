# Backend

API Node.js + TypeScript com NestJS, preparada para crescer de forma modular.

## Estrutura prevista

- `src/modules/` para módulos de domínio.
- `src/common/` para utilitários e peças compartilhadas.
- `src/config/` para configuração.
- `src/types/` para tipos do backend.

Nesta etapa, a pasta existe apenas como base organizacional.
